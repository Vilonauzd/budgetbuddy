Device Type Hours Formula Example
Windows 10/11 devices 0.5 hours/device Number of devices * 0.5
Route Switch and Firewall Devices 1 hour/device Number of devices * 1
Wireless Access Points and Controllers 0.25 hours/device Number of devices * 0.25
Windows Server Instances 0.5 hours/device Number of servers * 0.5
Hypervisor host instances 1 hour/device Number of instances * 1

Total Hours Calculation

Total Hours = Sum of (Number of devices * Hours per device)

Total Price Calculation

Total Price = Total Hours * Hourly Rate

Report Generation Details

CSV Report includes: Customer Name, Device Types and Counts, Total Hours, Total Price

HTML Report includes: Customer Name, Device Types and Counts, Total Hours, Total Price

Email Report includes: Customer Name, Total Hours, Total Price

Note: Net-New CMS Customers (with box checked) add +2 additional hours for each input section with a value above 0.
